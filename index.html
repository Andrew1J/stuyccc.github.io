Stuyvesant Competitive Computing Club¶

This site contains Stuyvesant Competitive Computing Club's lessons and resources. New lessons will be added to the site, with a recording of each meeting uploaded to the club's YouTube account.

Lessons¶

  1. Time and Space Complexity
  2. Data Structures Overview
  3. Sorting Overview
  4. Introduction to Graphs - A Node to Joy
  5. Breadth of the Wild: Breadth First Search
  6. Adjacency Lists and BFS Problems
  7. Greedy Algorithms - If Instant Gratification Worked
  8. USACO Practice

Lesson pages have runnable python fragments that can be interacted with. Be sure to follow along with the lesson by tinkering with the associated code fragments, and running them by using Shift-Enter.
