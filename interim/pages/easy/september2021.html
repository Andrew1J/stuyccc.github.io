EASY SEPTEMBER 2021 Problem 30
Solution

The optimal strategy for this problem is to make x = l, because the smaller x is the more multiples of x(ie x*2,x*3,x*4) that can be found the in range [l,r]. Now that we have x = l, the smallest multiple of x that is in the range and isn't x itself (x can't be lcm(x,y), because y > x from the problem) is 2*x. Since this is the smallest value of multiple possible, if this value is in the range we can just return x and 2x because the lcm of (x,2x) = 2x, However, if 2x is not in the range, then no other multiple of x is possible, and so just return -1 -1.


Problem 29
Solution

If an the apartment number we receive as input has size n and made of the digit d, then for all x < d we add 10 because we know that all apartments before the digit d were rung, so we add a total of 10 digits for every number x < d. And once we reach d, we know that all apartments with size 1, 2, ..., m were rung, so the total number of digits is n(n+1)/2. We also know that all apartments that come after this point wont be rung. So we get a final formula of (d-1)*10 + n(n+1)/2.
