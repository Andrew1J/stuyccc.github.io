Input And Output

Python

IO in python comes down to using the input() function and using python's string parsing.

input() returns a string containing one line of input from standard in. Standard in is one way a program communicates with the enviorment around it. When you run a program, input() will cause the program to wait until a line of text is found in standard in.

If you are just running a python program in terminal with the command python program.py, it will wait for you to type some characters and then press enter (or return, or whatever will input a new line). At that time, input() will return the string of characters you typed. 


s = input()
# run the program and type "hello"
# then press enter/return/whatever the equivalent for you is
print(s)
# hello is printed as s is the string "hello"

Now that you have a string, the most common things to do are to use strip() to get rid of trailing whitespace and split() if you want to turn a string of space separated elements of a list into a list. Use list/generator comprehension if you can.

I'll follow with writing some examples of python IO: 


# the input is in the format
# N
# where N is an integer
N = int(input())

# input() will read from standard in the string representation
# of N
# int() will parse that string for the integer N, and return
# that integer
            

# the input is in the format
# N M
# where N and M are both integers
N, M = [int(x) for x in input().split()]

# this uses input() to get the two integers in their string
# representation
# then split turns those integers into a list ['N', 'M']
# using list comprehension this turns the list [N, M]
# notice N and M are integers now
# finally these get set to the correct variables, N and M
            

# the input is a list of integers of format
# a1 a2 a3 ... an
# ai is an integer
A = [int(x) for x in input().split()]

# this is similar to the last example
# input() will get the string of space separated integers
# split() will turn that string into an array of strings
# representing integers
# using list comprehension and int
# we turn this array of strings into an array of integers

Final Note

Just using input() can be slow, so instead we use sys.stdin.readline. This doesn't do exactly the same thing as input() (it is one of those things to use for competitive programming but not in general) but for the use of competitive programming it is fine. This amounts to adding the following to the your program: 


import sys
input = sys.stdin.readline

Java

IO in Java (for competitive programming) generally uses the BufferedReader class along with the StringTokenizer class for input and simply System.out.print/System.out.println for output.

An example of reading in and then printing out an integer follows: 


//needed imports
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class A {
    public static void main(String[] args) throws IOException {
        //input example:
        //1
        BufferedReader in = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(in.readline());
        System.out.println(n);
    }
}

An example of reading in a list of integers with a given length is: 


//needed imports
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class A {
    public static void main(String[] args) throws IOException {
        //input example:
        //3
        //1 2 3
        BufferedReader in = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(in.readline());
        StringTokenizer st = new StringTokenizer(in.readline());
        int[] nums = new int[n];
        for (int i = 0; i < n; i++) {
            nums[i] = Integer.parseInt(st.nextToken());
        }
    }
}
