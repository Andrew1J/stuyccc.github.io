Why C++?

C++ is uber fast, relatively flexible, and has a large standard library. Even if that doesn’t make sense, just know that C++ is the most popular coding language used in competitive programming for very good reasons. So if we are going to teach you guys a coding language, why not teach the best of the best.

Function Definition

(Math) Accepts inputs, produces a unique output for each set of inputs.

(English) The ability to do something.

(Programming) A function is a part of code that just does something; it doesn’t necessarily need inputs or outputs but it can have both. These functions will contain algorithms. You don’t have to fully understand how to create functions yet!

Algorithms

An algorithm is a series of well defined steps that can be followed to produce a result. The following are some examples of algorithms

The Multiplication Algorithm

When you do long multiplication on paper you are actually using an algorithm. Consider 999 * 154.

To multiply these two numbers we first take the 4 and then multiply it with all the digits of 999, then we shift to the left and take the 5 and multiply it against all the digits of 999, and so on.

Problem A: Given a list of numbers A, find the range of A using an algorithm

Solution: Go through A once to find the maximum, then go through A again to find the minimum, and find the difference between the two.

Problem B: Given 2 lists of numbers A and B, determine whether an element in A and an element in B sum to some number k.

Solution: For each element a in A, for each element b in B, if a + b = k, it is possible. At the end of the process, if we have found no valid pairs, then it is not possible.

The explanations for each of these three problems above are in what’s called pseudo code: the explanation of an algorithm in English. It can then be translated into code.

Creating Hello World

You need two things to start writing C++ files. The first is a text editor, an application that allows you to write text to files.. Some commonly used ones are Atom, Vim, VSCode, Notepad++, and Gedit. It doesn’t matter what you use as long as you’re comfortable with it. Secondly, you need access to a terminal, which every computer has.

The code for your first file, which is typically Hello World (copy and paste this into your editor and save it):


#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << “Hello World!\n”;
}

The first line “includes” a large file with prewritten functions. Programmers are lazy! They don’t want to rewrite commonly used functions over and over again. The bits/stdc++.h file is included in the C++ compiler we suggest you to use.

This include file has a HUGE number of functions—the names of these functions could easily overlap with those from other libraries. Thus, another level of organization we use is namespaces. The second line identifies that we’re using a specific family of functions: in this case, functions from the standard library.

`int main()` is always the first function that’s called when the program is run. It is also the computer’s only point of entry—all other functions must be called from main. Thus, this is where we put the contents of our code.

The one line we have inside the main program is just an instruction to print “Hello World!”. We will talk more about printing in the future.

Running Programs

To run a program, we first need to download and use a compiler (typically g++, for C++ files) to translate the code you’ve written into a language that’s unreadable to us, but legible to a computer. This is put into a file called a.out. Running that file will execute the code. The terminal commands to execute code are g++ fileName.cpp and ./a.out

Where fileName is what you named your file. It is important that it is saved with the .cpp file extension. When that’s done, then viola! You’ve made your first C++ program.
